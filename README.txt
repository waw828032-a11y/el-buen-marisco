EL BUEN MARISCO - SISTEMA WEB FUNCIONAL

Qué hace:
- Meseros: toma pedidos por mesa
- Cocina: recibe pedidos enviados y los marca listos
- Caja: cobra mesas, imprime cuenta, lleva caja
- Reportes: ventas del día, productos más vendidos, control de meseros

Cómo usar en Windows:
1. Instala Python 3 si no lo tienes.
2. Abre la carpeta del proyecto.
3. Haz doble clic en run_app.bat
4. Se abrirá el sistema en: http://127.0.0.1:5000

Para usarlo en tablets o en otra computadora de tu mismo WiFi:
1. Ejecuta el sistema en la computadora principal.
2. Busca la IP local de esa computadora, por ejemplo 192.168.1.20
3. En las tablets abre: http://192.168.1.20:5000
4. Todos deben estar conectados al mismo WiFi.

Notas:
- La base de datos es el archivo restaurant.db
- Si quieres respaldo, copia ese archivo.
- Esta es una base funcional y se puede mejorar después con usuarios, contraseñas, inventario y respaldo en la nube.
